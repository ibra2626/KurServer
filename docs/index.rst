KurServer Documentation
========================

KurServer, Ubuntu sunucular için web sunucusu yönetim CLI aracıdır.

.. toctree::
   :maxdepth: 2
   :caption: Contents:

   user-guide/index
   developer-guide/index

Features
--------

* Nginx kurulumu ve yönetimi
* MySQL/MariaDB veritabanı kurulumu
* PHP-FPM çoklu versiyon desteği
* Site yönetimi (ekleme/silme/domain değiştirme)
* GitHub entegrasyonu ile proje dağıtımı
* Manuel proje dağıtımı

Installation
------------

.. code-block:: bash

   pip install kurserver

Quick Start
-----------

.. code-block:: bash

   kurserver
